## Requirements
```
Implement a Mutation which can register a new user with these properties
+ first name (required)
+ last name (required)
+ email (required and this property is unique)

- DTO
- Account Service
- Account Resolver
```

```
Implement a Query which list all users with these properties
+ id (required)
+ first name (required)
+ last name (required)
+ email (required)
```
